Movie Trailer Website

This project was created using a mix of Python and HTML/CSS. When you run the entertainment_center.py file, it will automatically generate a website and open it in a new tab on your machine. The website has a list of my six favorite movies, and has interactions that allow you to click on a movie portrait, and then view its YouTube video.
